# Knowledge Check

What are three reasons why you may see a TypeError?
- The function doesn't use the operand or type of input
- Modifying immutables
-  attempting to use a value in an inappropriate way.


What is the key difference between an error and a warning?
- Errors will stop the program/ process
- Warnings are likely informational and not as significant, and probably won't crash the program at runtime
  
What is one method you can use to resolve an error?
- Debugger
